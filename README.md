## 📄 Sobre o Projeto

A **Agenda de Contatos** é uma aplicação desenvolvida em Python com o objetivo de gerenciar contatos de forma simples e eficiente. Este projeto foi desenvolvido durante os estudos na trilha da **Rocketseat** para aplicar conceitos fundamentais de programação, como manipulação de dados, operações de CRUD (Create, Read, Update, Delete) e lógica estruturada.

É uma ferramenta prática e direta para cadastrar, buscar, editar e excluir informações de contato diretamente pelo terminal.

## ✨ Funcionalidades

- **Adicionar Contato:** Permite salvar um novo contato.
- **Listar Contatos:** Exibe todos os contatos salvos na agenda de forma organizada.
- **Buscar Contato:** Filtra e encontra um contato específico.
- **Editar Contato:** Atualiza as informações de um contato já existente.
- **Excluir Contato:** Remove um contato permanentemente da agenda.

## 🛠️ Tecnologias Utilizadas

- **[Python 3.x](https://www.python.org/):** Linguagem principal utilizada para a lógica da aplicação e execução no terminal.
